End-To-End Flow

1. SimulateGenerate facility metrics and scenario profiles.
2. PublishSend metric payloads to MQTT topics.
3. IngestNormalize and enrich telemetry rows.
4. StorePersist time-series data in ClickHouse.
5. AnalyzeEvaluate alert rules and maintenance risk.
6. ExposeServe APIs and Prometheus metrics.
7. ObserveRender Grafana dashboards.
8. OperateUse React for scenarios and alert actions.
